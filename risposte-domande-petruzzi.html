Risposte alle domande:

1) Software Git - è un software di controllo della versione del codice distribuito, lo usano i developers per tenere i file sempre aggiornati in un team di lavoro su uno spazzio virtuale chiamato "Repository Remoto".
2) Ambiente di lavoro:
La tua copia locale del repository è composta da tre "alberi" mantenuti da git. Il primo è la tua Directory di lavoro che contiene i files attuali. Il secondo è l'Index che fa da spazio di transito per i files e per finire l'HEAD che punta all'ultimo commit fatto.
3) Branch - significa "ramo", un repository remoto puo contenere diversi versioni del progetto, oppure dei file ulteriory per un file completto.
4) Il Branch(ramo) - si stacca dall'albero principale. L'albero principale(main) - si chiama "master".
5) Commit - grazie a Git, ogni volta che si lavora su un progetto e si fanno dei cambiamenti a dei file o alla struttura è possibile dare un ID, un nome e una descrizione a questo set di cambiamenti. Questa cosa si chiama commit.
